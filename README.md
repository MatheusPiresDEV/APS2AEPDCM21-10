Descrição dos Diretórios
Cada pasta representa um conjunto de exercícios com foco em um tema específico da linguagem C++. A estrutura interna de todas elas é padronizada:

Arquivos .cpp
Cada arquivo representa um exercício individual. Os códigos seguem uma estrutura comum, com foco em entrada de dados, processamento e saída. Os temas abordam lógica de programação, estruturas de repetição, vetores, matrizes, modularização e estruturas.

Subpasta Executaveis/
Contém os arquivos compilados (.exe) correspondentes aos exercícios da pasta principal. Cada executável é gerado a partir de seu respectivo arquivo .cpp, mantendo a mesma nomenclatura.

Instruções de Execução
Abra o projeto no Visual Studio Code ou outro ambiente de desenvolvimento compatível.

Navegue até o diretório do exercício desejado.

Compile o arquivo .cpp utilizando um compilador C++:

bash
g++ EX01.cpp -o Executaveis/EX01.exe
Execute o arquivo gerado:

bash
./Executaveis/EX01.exe
Observações
Todos os diretórios seguem a mesma estrutura e são independentes entre si.

Os arquivos seguem nomenclatura padronizada (EX0X.cpp e EX0X.exe).

O projeto foi organizado para facilitar o aprendizado modular e progressivo da linguagem C++.
